# koa-initbase

一个使用 koa2 构建的 mvc 设计结构，直接使用 JavaScript 编写

> 基于 Windows 平台的构建、开发和测试，因此可能在别的平台上还存在一些问题
>
> Windows: 推荐 Windows 10 及以上版本
>
> **该项目为笔者学习参考使用**



#### fetch

```
git clone https://github.com/ctime-zliang/koa-initbase
```



#### usage

```
// 具体以 package.json 中定义的命令为准

// 以开发模式启动服务
npm run server:dev

// 使用 PM 执行服务
npm run server:pm:*
```
